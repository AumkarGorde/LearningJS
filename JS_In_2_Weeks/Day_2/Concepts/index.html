Limitations

Not suitable for methods with their own 'this' context:

Arrow functions capture the 'this' value from their lexical (surrounding) context when they are created, not when they are executed.
Arrow functions are particularly useful when you want to capture the this value from their surrounding context. This is beneficial when working with functions inside object methods or when you need to maintain a consistent this context within a closure.

'this' refers to the current object or context.
1. In regular functions, this can change depending on how the function is called.
2. Arrow functions capture the this from the surrounding context, making it predictable and useful for maintaining the same context.
So, when writing JavaScript code, you need to consider how this is used and whether you want it to stay the same or adapt to different contexts. Arrow functions are a handy way to ensure this behaves consistently within a particular scop

Destructuring Objects
Destructuring is a feature in JavaScript that allows you to extract values from objects and arrays and assign them to variables in a more concise and readable way.
It simplifies working with objects and arrays by reducing the need for manual property or element access and assignment.
Destructuring can be used with objects and arrays, and it greatly improves code clarity and maintainability.
